## Developing

To set up your local environment, run the following:<br>
`yarn install`<br>
`yarn start` <br>

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### For ESLint's and Prettier lint check,

    yarn lint

### For ESLint's and Prettier lint fix,

    yarn lint-fix
